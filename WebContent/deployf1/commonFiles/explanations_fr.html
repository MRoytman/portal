Introduction

Vous trouverez ci-dessous toutes les informations necessaires pour gérer la collecte de données. La collecte des données patient servira à l'établissement des rapports de statistiques. Lire avec attention toutes les informations de cette page pour comprendre les opérations à éffectuer.

Prérequis d'installation

  • Avant de lancer le logiciel de saisie de données, assurez vous d'avoir reçu le fichier de configuration "entryFormConfig.xml". Sans ce fichier le logiciel ne pourra pas démarrer. Celui-ci doit vous avoir été fourni par un manager au niveau de la coordination MSF et doit être installé à l'emplacement suivant:
    "D:\".
    Note: L'ordinateur doit donc posséder un disque D et disposer de suffisament de place (au moins 5Mb).

Les outils à disposition permettent les actions suivantes:

  • Saisir des informations de patients et de consultations dans la base de donnée.
  • Exporter des données de la base de donnée vers le 'Head Quarter'.
  • Consulter des statistiques.

Guide rapide d'utilisation

  • Pour lancer le logiciel de saisie des données, cliquer sur le bouton 'Collecte des données'.
  • Pour lancer le logiciel d'export des données, cliquer sur le bouton 'Export des données'. L'export des données est un passage obligé pour produire les statistiques.

Terminologie/Définitions

  • Console java: Une fenêtre dans laquelle les messages système sont envoyés.
    Note: Cette fenêtre apparait seulement si l'option 'show console' a été configurée dans la partie java du 'control panel' de Windows. Donc si elle apparait, cette fenêtre peut être minimisée, mais ne doit pas être fermée car elle peut être utilisée pour communiquer des messages d'erreur potentiel.
  • MSF application: Tout logiciel MSF qui est lancé à partir d'un des liens accessible sur ce site.

Installation ou démarrage d'une application MSF

  • Avant le démarrage: Il faut s'assurer que l'application (ainsi que la console java) n'est pas déjà lancée. Si c'est le cas, il faut (les) l'arrêter.
  • Installation ou démarrage: Le fait de cliquer sur un lien démarre l'application. La première fois, cela va également installer le logiciel et mettre un raccourci sur le bureau de l'ordinateur. Lors du lancement plusieurs boites de dialogue apparaitront successivement. L'une d'entre elle, intitulée "Warning-Security" vous demandera la confirmation du lancement de l'application. Avant de cliquer sur le boutton "Run", cochez la case "Always trust content from this publisher".


    Lorsque le logiciel est installé, la fenêtre de l'application apparaitra après quelques minutes (selon la puissance de la machine et de la connectivité). Si la fenêtre de la console java est autorisée, elle apparaitra également.
  • Mots de passes: Les applications MSF 'sensibles' sont protégées par mots de passe. Un mot de passe vous sera communiqué si vous êtes autorisé à utiliser l'application en question.
