Introduction

You will find below all the necessary information to manage the data collection. Patient data collection is needed to run statistics reports. Please read carefully this page to get familiar with the concepts.

Installation prerequisites

  • Before running the data entry form software, make sure you have received the configuration file named "entryFormConfig.xml". This file must have been sent to you by a coordination manager prior to installation (the software won't work otherwise). and must be located under the directory:
    "D:\".
    Note: It should be noted that the computer should have a drive named D with sufficient space (5Mb is a minimum requirement).

The provided tools allow to:

  • Enter patients and consultations data in the database.
  • Export data from the database to the 'Head Quarter'.
  • Show statistics.

Quick user guide

  • To start the data entry form software, click on the 'Data entry form' button.
  • To start the export software, click on the 'Data export' button. Exporting the collected data is a mandatory step to build the statistics.

Terminology/Definitions

  • Console window: A window in which system messages are printed.
    Note: The console window only appears if it has been configured as 'show console' in the java part of the control panel. If the console window appears on screen, it can be minimized, but not closed as it can be used to communicate potential error messages.
  • MSF application: Any application that can be started with one of the links in paragraph below.

Installation or start of an MSF application

  • Before starting: Make sure the application (nor its java console) is not already started. If it is the case, kill the window(s) first.
  • Starting an MSF application: Once you have clicked on any of the links proposed below, you will see a serie of dialog boxes appearing on screen. One of them entitled "Warning-Security" will ask you to confirm the run. Before clicking on the "Run" button, check the "Always trust content from this publisher".


    At the time the software is installed, after few minutes (depending on the power of the running machine and connectivity), the application windows is displayed. If the java console is enabled, it will appear too.
  • Passwords:Sensitive MSF applications are password protected. If you are entitled for using one of these, you will be given a user&password information.
