MSF Application links

  • Install Click on this link to setup your machine for reporting. The software will automatically install components necessary to run the 'Analysis' software. You must use this link at each new 'Analysis' version.
  • Admin Click on this link to define the scoping contexts (countries, projects, care centres) that will be used for the data collection software 'EntryForm'. This process has to be done prior using the data collection application.
    Usually this operation is done at the coordination level, but in certain cases it is done at the field level.
    The output of this process is the generation of a configuration file named "d:\entryFormConfig.xml". This file should be copied to the field data operator machine where the 'EntryForm' software will be run.
    For more information consult the Admin User Manual.
  • EntryForm Click on this link to perform the data collection application. Prior launching the application, you must ensure the file named "d:\entryFormConfig.xml" is installed on your machine (this file contains at least one context information and SHOULD NOT be edited manually).
    For more information consult the EntryForm UserManual.
  • Analysis Click on this link to run the reporting application.
  • Export Click on this link to run the data export toward a central repository.
